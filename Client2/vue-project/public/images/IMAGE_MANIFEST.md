## How to Download

### Option 1: Using PowerShell Script
Run the `download-images.ps1` script to automatically download all images.

### Option 2: Manual Download
1. Open each URL in your browser
2. Right-click and "Save image as..."
3. Place it in the corresponding folder

### Option 3: Using curl or wget
```bash
# Example: Download hero image
curl "https://images.unsplash.com/photo-1566073771259-6a8506099945?w=1920" -o hero/hero.jpg
```
